# Stacked Copy Paste
VSCode extension. Copy and paste multiple text using stack.
![](./README/000.gif)
## Copy `Ctrl+Alt+C`
Push selected text to stack.
## Cut `Ctrl+Alt+X`
Push selected text to stack and delete it.
## Paste `Ctrl+Alt+V`
Pop text from stack and insert.
## Undo `Ctrl+Alt+Z`
Undo stack manipulation.
